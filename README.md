# Sistema de Reportes de Precios

## Objetivo general
Desarrollar un sistema en Python que genere reportes automáticos de precios a partir de la base de datos de SQL Server, con el fin de apoyar la toma de decisiones en el área de precios.

---

## Primer reporte Análisis del Margen de Utilidad de los pedidos afectados diarios 

Este reporte servirá como base para futuros cambios en la aplicación de descuentos y mejorar los margenes de ganancia 

##  Fuente de datos
- Base de datos principal: **SQL Server**
- Acceso futuro mediante conexión desde Python (pyodbc o SQLAlchemy)

- Planeación
